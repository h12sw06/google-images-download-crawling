#사용법
1. pip install
pip install bs4
pip install selenium

-----
2. 원본.py 또는 test copy.py를 실행 하시면 됩니다.

#참고
1-1. 원본.py 또는 test copy.py에서 searchItems에 찾고 싶은 키워드를 입력하시면 됩니다.
1-2. 원본.py는 리스트형태로 여러가지 키워드를 지정 할수 있습니다.

2-1. 끌어온 이미지 파일은 image에 키워드형태의 폴더이름이 생성 되며 그안에 이미지들이 있습니다
2-2. 제가 한번 테스트 차원으로 "휠체어 무료대여" 폴더가 있습니다.
